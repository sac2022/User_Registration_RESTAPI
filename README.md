POST https://immense-cliffs-30014.herokuapp.com/verify

**Request Body**:
```
{
      "email":"sac2023@gmail.com",
      "otp":"836428",             
      "user_id": "5"              

}
 ```
**Response**
`Success`:
 HTTP:200
```
{
  "msg": "user verified"
}
```
`Error`
 HTTP:500
```
 {
    "msg": "invalid otp"
 }
```
 `Error`

 ```
 {
   "msg": "User Doesnt exist"
  }
```   
